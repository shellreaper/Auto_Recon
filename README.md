# Auto_Recon
Bash script to automate bug bounty hunting reconnaissance phase

### Installation:
```bash
chmod +x setup.sh
./setup.sh
```


### Usage:
```bash
./auto_recon.sh example.com example_dir 
```

### Tools required(run setup.sh to install them all):
- amass
- subfinder
- nuclei
- naabu
- puredns
- dnsgen
- httpx
- assetfinder
- Seclists( in /usr/share/)
